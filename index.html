O que é

HTML5
Html (hyperText Markup Language" ou "linguagem de marcação de hiperTexto) é uma linguagem de marcação, seria algo como o esqueleto de um site, onde sera a base para as outras partes que compõem um web Site,o Html Funciona em um modelo de 'Tags' onde cada tag existe para um fim. exemplo: se quisermos um texto como esse Usamos a Tag 'p' que é para paragrafos. é nele tambem que configuramos a nossa pagina em relação ao tipo de dispositivo idioma e links externos para fontes e arquivos diversos. veja mais aqui
Css3
Css ('Cascading Style Sheets' ou folha de Estilo em cascata) é onde estilizamos nossa pagina, onde definos que tipo de tipografia, cores, tamanhos, bordas e uma infinidade de estilizações usaremos. seguindo a logica de representar o html como um esqueleto o css pode ser visto como nossos estilos (cor da pele, cor e tipo do cabelo, tamanho das orelhas, etc...). veja mais aqui
JavaScript
com ele podemos deixar nossas paginas inteligentes como manipular eventos, posições, tomar desições de acordo com comportamentos dos usuarios, etc... manipulamos com js o que Chamamos de "DOM". na analogia de um corpo o js seria os musculos e cerebro humano dando infinitas posibilidades. sendo que sem ele nao teriamos quase nada em relação a web que temos hoje. veja mais aqui
SCSS
SASS é um pré-processador de css, usamos sass para fazermos nossos estilos de forma mais simples, aplicando conceitos de programação, com ele modularizamos nosso codigo e o deixamos mais facil de se ler e manipular depois de um tempo, ele é de uma forma grosseira uma ferramenta da ferramente. veja mais aqui
